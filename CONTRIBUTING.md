# Contributing to MoltLock 🛡️

First off, thanks for taking an interest in AI Agent security. **MoltLock** is an open-source protocol designed to solve the "Agency Liability Crisis." 

### 🤝 How to Contribute
1. **Bug Reports:** If the Judge hallucinations or the Handshake fails, please open an Issue with your terminal logs.
2. **Feature Requests:** We are currently looking for:
   - Local LLM Judge support (Llama 3 / Mistral).
   - Discord/Slack notification adapters.
   - Hardware key integration (YubiKey).
3. **Pull Requests:** - Please create a feature branch (`git checkout -b feature/cool-new-thing`).
   - Ensure your code follows the existing ANSI color logging format.
   - All PRs must be reviewed and signed off by @berkmh.
